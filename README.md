# MyBus Nancy UX Heuristic Evaluation

## Overview

This project presents an ergonomic and usability evaluation of the **MyBus Nancy** mobile application using **Jakob Nielsen's 10 Usability Heuristics**.

MyBus Nancy is a mobile application available on Android and iOS that provides users with quick access to public transportation information in Nancy and surrounding areas. The application offers bus schedules, route planning, service disruption alerts, and real-time transportation updates.

The purpose of this evaluation is to identify usability strengths and weaknesses and propose improvements to enhance the overall user experience.

## Evaluation Method

The analysis is based on **Jakob Nielsen's 10 Usability Heuristics**, a widely recognized framework for evaluating user interfaces and improving user experience.

### Nielsen's Heuristics

1. Visibility of System Status
2. Match Between System and the Real World
3. User Control and Freedom
4. Consistency and Standards
5. Error Prevention
6. Recognition Rather Than Recall
7. Flexibility and Efficiency of Use
8. Aesthetic and Minimalist Design
9. Help Users Recognize, Diagnose, and Recover from Errors
10. Help and Documentation


## Objectives

* Evaluate the usability of the MyBus Nancy application.
* Identify ergonomic issues based on Nielsen's heuristics.
* Assess the effectiveness of the interface design.
* Provide recommendations for improving the user experience.
